Using the Problem Solving Skills in Programming Fundamentals

Although I had some knowledge when it came to programming in the language of Python, before coming into the semester, I have never considered implementing specific problem solving strategies when programming or solving a problem using code. In general, I would attempt to solve my problem in a Brute force type of approach. However, later on the way in my programming journey, I realized that this method is not always going to work in favour of mine and there are better more efficient/effective ways to go about approaching a problem of code. And that is when I considered looking into the different problem solving strategies. Some of the strategies that I have learned were Decompostion, which is breaking down components of a problem into smaller steps, Abstraction, making an "Abstract" view of the problem to better understand it, and Analogy, which is applying something that I am familiar with from before, in order to solve the problem.
